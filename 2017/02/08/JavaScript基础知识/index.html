函数本身的作用域

函数本身也是一个值，也有自己的作用域。它的作用域与变量一样，就是其声明时所在的作用域，与其运行时所在的作用域无关。

闭包

由于在JavaScript中,只有函数内部的子函数才能访问内部变量,因此可以将闭包简单理解成”定义在一个函数内部的函数”.
闭包最大的特点,就是它可以”记住”诞生的环境,比如f2记住了它诞生的环境f1,所以f2可以得到f1的内部变量.
本质上,闭包就是将函数内部和函数外部连接起来的桥梁


闭包最大的作用有两个,一个是读取函数内部的变量,另一个就是让这些变量始终保持在内存中,
即闭包可以使得诞生环境一直存在.下面例子使得内部变量记住上次调用的运算结果
